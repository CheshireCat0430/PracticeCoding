## 깃허브와 연결방법
> + 깃허브에 공간 만들기
> + 작업공간에 깃허브 설치
>   + vscode
>   ```
>   sudo apt-get install git
>   ```
>   + wsl
>   ```
>   # add-apt-repository ppa:git-core/ppa 
>   # apt update
>   # apt install git
>   ```
> + 깃 복사하기
> ```
> # git clone <깃허브 레포 주소>
> # git clone https://github.com/Forenser0/network.git
> ```
> + 폴더 만들어서 작업 폴더로 설정하기
> + 초기화 및 초기 세팅하기
> ```
> # git init
> # git config --global user.name "Forenser0" 
> # git config --global user.email vickey043011@gmail.com
> ```

## 깃허브 사용하기
> + commit 방법
> ```
> # git add .
> # git commit -m "커밋 내용"
> # git push
> ```
> + pull 방법
> ```
> # git pull
> ```
